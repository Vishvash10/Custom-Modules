Manufacturer

Manufacturer is module where user able to store manufacturer details in odoo.

In this module by cron job automatically manufacturer data fetched from api and store it in records.

In this module user also able to print manufacturer data(detail).


Installation

To install this, follow below setps:

  • Just simply mount this module as Odoo's custom module

  • Now, Install the module in Odoo from Main Apps section.


Usage

Follow the below steps before using this module

  • Open Odoo -> Go to Apps -> Manufacturer application

  • Click on Create -> Enter following details related to manufacturer -> Save Your data is stored.

  • Now able to see your List View and Form View of your data.

  • In this application go in Products menu -> click on products sub menu -> click on any product. You see a option Manufacturer in product form.

  • Click on edit select Manufacturer now enter Manufacturer Details and Save.

  • In Manufacturer module you can able to print Manufacturer Data(detail). Go in manufacturer module list view -> Click on print -> click on print manufacturer detail -> wait some time -> your data is stored in PDF format.

  • In Manufacturer module you can also able to print all Manufacturer Data(detail) in one time. Select all record from tree view then -> Click on print -> click on print manufacturer detail -> wait some time -> your data is stored in PDF format.


Dependent modules

Features

  • This module able to Store Manufacturer Details provided by user.

  • Lots of information stored related to Manufacturer. e.g. Company information, Contact information

  • Provide new tab in product form.Where user able to add / update / delete manufacturer details.

  • In this module you are able to fetch data from API. By cron job in every hour manufacturer data fetched from API and save in records.

  • In this module you are able to print Manufacturer Data(details). And this report also able to print data in user specified language(If translation is available).

    • User able to print data in two ways.

    • 1. One manufacturer data at one time.

    • 2. All manufacturer record (data) at one time.



Quick Guide



  • After installation -> Click on menu -> Click On Manufacturer application





  • You see list view if any data present other wise you see blank list view.





  • Click on create to create new manufacturer details





  • Click on list view if available





  • Form view opened of manufacturer details





  • Click on any manufacturer detail. Then click on print then click on print for "Report".





  • In tree view select all manufacturer data then click on print then click on print manufacture data for "Report".
